#MinerPool

虚拟货币三种常用的矿池挖矿方法总结脚本：

1 CPU 挖矿 minerd

sh minerd_linux.sh
minerd_win.cmd
sh minerd_mac.sh


2  A卡挖矿
amdminer_win.cm   //cgminer
sgminer_win.cmd


3  N卡挖矿

cudamin_win.cmd



4. MAC 电脑上挖矿软件

MacMiner.app-1.5.42.zip   MacMiner需要解压缩安装


在使用以上脚本时，需要修改矿池的地址和矿工
比如：minerd.exe --timeout=6 --url=stratum+tcp://stratum.aikapool.com:7912 --userpass=aohas.aohas7:888888
